# Sample Requirements — E-Commerce Platform
## Functional Requirements
- Req-1: Users must be able to register with email and password.
- Req-2: Users must be able to log in with Google OAuth.
- Req-3: Users must be able to log in with GitHub OAuth.
- Req-4: The product catalog must support filtering by category, price range, and rating.
- Req-5: Users must be able to add items to a shopping cart and modify quantities.
- Req-6: Users must be able to checkout using credit card or PayPal.
- Req-7: The system must send order confirmation emails after successful payment.
- Req-8: Admin users must be able to manage products (create, update, delete).
## Non-Functional Requirements
- Req-9: The product catalog page must load in under 2 seconds.
- Req-10: The system must support at least 1000 concurrent users without degradation.
- Req-11: All user passwords must be stored using bcrypt with salt.
- Req-12: The system must comply with GDPR for EU users.
