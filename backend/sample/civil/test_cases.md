# Sample Test Cases — Highway Bridge Structural Design

## Uploaded Test Cases (partial coverage)

- Test-2: Measure girder deflection at midspan under maximum service load — verify deflection does not exceed L/800.
- Test-3: Perform nonlinear time-history analysis using 5.0 magnitude earthquake record — verify ductility ratio >= 3.0.
- Test-4: Conduct load test on completed pier column — verify ultimate bearing capacity meets 300 kPa with safety factor 2.5.
- Test-5: Perform material coupon testing on delivered steel — verify ASTM A709 Grade 50W compliance.
